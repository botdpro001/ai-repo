# CyberShield HER - AI Fake Social Media Account Detector

Protect yourself from fake social media accounts, impersonation, scams, and identity fraud with AI-powered detection. Especially designed for women's online safety.

## Project info

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

To run this project locally:

1. Clone the repository
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm run dev
   ```
